# Amnesiac
Crossplatform FOSS (Free and Open Source Software) GUI for AmneziaWG protocol, written in Flutter.

> [!WARNING] Project in early development now, there're a lot of bugs, unimplemented functions and other problems.

## Features

- Full support of all main platforms;
- Split tunneling;
- Modern UI, based on Material 3 Design (with dark theme and color changing (planned));
- Toggle to classic WireGuard


## Building

### Windows

`flutter build windows --release`

### Linux

`flutter build windows --release`

### MacOS

`flutter build windows --release` (I have no opportunity to test this platform)

### Android

`flutter build windows --release`

### IOS

`flutter build windows --release` (I have no opportunity to test this platform too)
